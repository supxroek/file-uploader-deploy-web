# file-uploader-deploy-web

## Setup Instructions

### Server Setup

1. Navigate to the server directory:

   ```sh
   cd /path/to/server
   ```

2. Install the dependencies:

   ```sh
   npm install
   ```

3. Start the server:
   ```sh
   npm start
   ```

### Client Setup

1. Navigate to the client directory:

   ```sh
   cd /path/to/client
   ```

2. Install the dependencies:

   ```sh
   npm install
   ```

3. Start the client:
   ```sh
   npm start
   ```
